# Explorando a Sintaxe: Operações Matemáticas em Portugol

Exercícios de lógica de programação focados em cálculos matemáticos, desenvolvidos para prática acadêmica utilizando Portugol.

## Exercícios

1. **Fechamento de Caixa**  
Calcula o valor total de um produto com base no preço unitário e na quantidade comprada.

2. **Conversor de Temperatura (Fahrenheit)**  
Converte uma temperatura em Celsius para Fahrenheit utilizando a fórmula de conversão.

3. **Média Ponderada**  
Calcula a média ponderada de duas notas com pesos diferentes.

4. **Consumo do Automóvel**  
Calcula o consumo médio de combustível de um veículo em km/l.

5. **Cálculo de Desconto Profissional**  
Calcula o valor de desconto aplicado a um produto e exibe o novo preço final.
